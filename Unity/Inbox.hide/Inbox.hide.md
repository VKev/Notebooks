# Inbox

## Purpose
- Folder `Inbox.hide/` chứa `fleeting notes` để capture nhanh ý tưởng, câu hỏi, hoặc observation chưa xử lý.
- Các note trong `Inbox` không cần hoàn chỉnh.

## Workflow
1. Capture nhanh ý tưởng vào `Inbox.hide/`.
2. Review lại sau.
3. Nếu note có giá trị lâu dài, rewrite nó thành `permanent note`.
4. Đặt `permanent note` vào đúng father folder.
5. Link note đó từ father note tương ứng.

## Naming
- Fleeting notes có thể dùng timestamp hoặc tên ngắn để capture nhanh.
- Sau khi rewrite thành `permanent note`, file name nên quay về English semantic name.
