# Templates

## Purpose
- Father note này gom các template dùng để tạo note mới nhanh và nhất quán.
- Đây là system area, không phải topic knowledge area.

## Notes
- Permanent Note Template, Permanent Note Template: template cho child note kiểu `permanent note`.
- Structure Note Template, Structure Note Template: template cho father note kiểu `structure note`.
- Fleeting Note Template, Fleeting Note Template: template cho `Inbox` capture.

## Obsidian setup
- Dùng core plugin `Templates` và đặt template folder là `Templates.`.
- Nếu muốn capture nhanh theo timestamp, dùng `Unique note creator` cho `Inbox.`.
- Sau khi capture, rewrite lại note có giá trị thành `permanent note`.
